Uma arquitetura clássica de 32 bits de espaço de endereçamento de memória permite até 4 GB de memória principal. Para um bom gerenciamento dessa memória, algumas técnicas podem ser utilizadas: paginação, segmentação e swapping.

Em relação a essas técnicas de gerenciamento de memória, avalie as afirmações a seguir.

  1. Utilizando páginas de 4 KB de tamanho, o endereço virtual de memória utiliza os 220 bits mais significativos para o número da página virtual e os demais 212 bits para o deslocamento interno da página.
  2. O acesso à memória física, utilizando segmentação, é feito por meio de uma estrutura de hardware específica do processador.
  3. Utilizando a paginação ou a segmentação, a memória física não sofrerá fragmentação interna ou externa.
  4. É possível utilizar a técnica de swapping para qualquer tamanho de página, mas não a segmentação, pois esta tem espaço de endereçamento variável.
  5. O algoritmo de substituição de página, denominado LFU, utiliza o princípio de localidade temporal, enquanto o algoritmo LRU utiliza o princípio de localidade espacial.
