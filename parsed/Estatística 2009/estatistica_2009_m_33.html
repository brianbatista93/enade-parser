O Estatístico consultado para resolver esse problema selecionou uma amostra aleatória de vendas, ajustou um modelo de regressão linear usando essas variáveis e apresentou ao gerente as conclusões:

  1. O gasto com acessórios cresce linearmente com o valor do automóvel.
  2. As mulheres gastam menos com acessórios do que os homens.
  3. O gasto com acessórios decresce linearmente com a idade do comprador.
  4. O efeito linear do valor do automóvel no gasto com acessórios não é o mesmo nos dois sexos.
  5. O efeito linear do valor do automóvel no gasto com acessórios não é o mesmo para qualquer idade do comprador (e vice-versa).
  6. O efeito linear da idade do comprador no gasto com acessórios é o mesmo nos dois sexos.
